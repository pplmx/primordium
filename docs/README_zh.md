## 🎯 愿景

Primordium 是一场关于**涌现复杂度**的实验。它将你的宿主机转化为数字神明：

- **CPU 负载** 转化为环境的气候压力。
- **内存占用** 控制资源的稀缺程度。
- **神经网络** 随着生物适应你硬件的独特运行特征而不断进化。

---

## 🌊 核心机制：硬件共鸣

模拟环境通过 `sysinfo` 实时获取计算机指标，并将其转化为生存压力。

### 🌡️ 气候 (CPU 耦合)

你的 CPU 使用率决定了所有数字生命的代谢速度。机器负载越高，生命消耗能量的速度越快。

| CPU 使用率 | 气候状态 | 代谢倍率 | 生存影响 |
| ----------- | ---------- | ---------- | ---------- |
| 0-30%     | 🌡️ 温和  | ×1.0    | 基准生存环境 |
| 30-60%    | 🔥 温暖  | ×1.5    | 能量消耗加速 |
| 60-80%    | 🌋 炎热  | ×2.0    | 高代谢压力 |
| 80-100%   | ☀️ 酷热  | ×3.0    | 大规模饥荒风险 |

### 🌾 资源稀缺 (RAM 耦合)

内存占用决定了食物的生成频率。高内存占用会模拟资源匮乏的饥荒环境。

## ✨ 功能特性

### 🧠 神经觉醒与并行智能

每个个体都拥有一个集成在 **Intel** 组件中的**动态图结构 (NEAT-lite)** 大脑。通过自然选择，生物会学会狩猎、导航和能量管理。

- **可进化拓扑**: 大脑不再是静态的。通过结构变异，生物可以生长新的神经元和连接，以适应复杂的生存环境。
- **效率压力**: 结构复杂性伴随着代谢成本（每个节点 0.02，每个连接 0.005），确保只有最有效的认知结构能留存。
- **循环记忆**: 6 个输入端专门用于保留上一时刻的内部状态，实现行为的连贯性。
- **并行推理**：利用 **Rayon** 实现多线程神经计算，支持超大规模种群模拟。
- **交互式选择**：直接使用 **鼠标** 点击屏幕上的生物进行锁定和深度检视。
- **基因型聚类**：系统根据神经权重相似度自动将生物归类为不同的"物种"。
- **分层帮助系统**：按 `H` 开启四页式帮助（控制、符号、概念、纪元）。

### 🦁 顶级掠食者与有性繁殖

- **捕食动态**：生物可进化出攻击性，捕食其他生物以获取巨大能量收益。
- **基因交叉**：有性繁殖支持与附近配偶进行神经特征交换。
- **HexDNA 协议**：通过文本文件导出（`C` 键）和导入（`V` 键）生物基因组。

### 📊 全知之眼

- **纪元系统**：由种群驱动的叙事引擎，追踪世界纪元（创世、扩张、衰落等）。
- **名人堂**：实时排行榜，展示适应度最高的前 3 名生物。
- **高级分析**：脑熵、平均寿命和双火花线种群动态。

### 🏔️ 生态动力学

- **地形系统**：山脉（▲减速）、河流（≈加速）、绿洲（◊食物丰富）
- **季节循环**：春/夏/秋/冬影响代谢和食物供应
- **地理压力**：基于地形的资源分布产生涌现式迁徙模式

### 👥 社会结构

- **信息素系统**：食物踪迹吸引觅食者，危险信息素警告掠食者
- **部落形成**：颜色相似的实体形成保护性部落（同部落不互相攻击）
- **领地行为**：实体在出生地附近更具攻击性
- **能量分享**：高能量实体可与饥饿邻居分享能量（♣）

### 🌌 全球蜂巢与联网 (阶段 22)

- **分布式演化**: 实体通过高性能的 **Axum** 中继服务器在模拟实例之间迁移。
- **跨宇宙移民**: 移出屏幕边缘即可将你最成功的谱系发送给其他用户。
- **实时同步**: 一个共享的数字多重宇宙，生命在这里跨越边界不断演化。

### 📜 演化账本与区块链

- **历史日志**：所有出生、死亡和环境事件实时流式记录为 JSONL 格式。
- **永恒证明**：演化数据集通过 SHA-256 哈希，并利用 OpenTimestamps 锚定至 **比特币区块链**，提供不可篡改的演化证明。

### ⚡ 性能与稳定性

- **基于组件的实体 (CBE)**: 逻辑分组属性为 Physics, Metabolism, Health 和 Intel，提升数据局部性。
- **系统化解耦**: 单体更新逻辑拆分为专业化系统（感知、行动、生物、社交）。
- **多核并行**: Perception 和 Neural 阶段完全并行化，极致利用多核性能。
- **严苛质量门禁**: 100% Clippy 通过率，40+ 单元与集成测试保障。
- **空间哈希 (Spatial Hashing)**: 优化后的感知算法，支持在大规模种群下流畅运行。

## 🚀 快速开始

确保你已安装 [Rust](https://rustup.rs/)。

```bash
# 克隆并进入目录
git clone https://github.com/pplmx/primordium.git
cd primordium

# 运行标准模式
cargo run --release

# 运行画境模式 (极简)
cargo run --release -- --mode screensaver
```

## ⌨️ 交互快捷键

| 按键 | 动作 |
| ----- | ------- |
| `Q` | 退出模拟 |
| `Space` | 暂停 / 恢复 |
| `B` | 开关神经网络热图 |
| `H` | 开关交互帮助菜单 |
| `X` | 触发 **基因风暴** (全球大突变) |
| `C` | 导出选中生物的 HexDNA |
| `V` | 从 HexDNA 文件注入生物 |
| `+` / `-`| 加速 / 减速模拟 |
| `鼠标左键` | 选择并追踪个体 |
| `鼠标右键` | 在点击处注入食物簇 |

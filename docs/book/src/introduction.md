## What is Primordium?

Primordium is an experiment in **emergent complexity**. It transforms your host machine into a digital god where:

- **CPU Workload** becomes the environmental climate
- **RAM Pressure** controls resource scarcity
- **Neural Networks** emerge through natural selection
- **Every legendary organism's DNA** is preserved on blockchain
- **Your machine becomes a god**, and you become the observer

Every run is unique. Every lineage is precious. Every extinction teaches us something.

## Quick Start

```bash
# Clone and run
git clone https://github.com/pplmx/primordium.git
cd primordium
cargo run --release

# Run in screensaver mode
cargo run --release -- --mode screensaver
```



### With Custom Configuration
```bash

```

## Performance Scale

| Entity Count | Tick Time (avg) | Recommendations |
|-------------|----------------|----------------|
| 100-500 | <5ms | Any hardware setup |
| 500-1,000 | <15ms | Modern multi-core CPU preferred |
| 1,000-5,000 | <50ms | Requires 4+ cores, 32GB+ RAM suggested |
| 5,000-10,000+ | <100ms | Needs 8+ cores, 64GB+ RAM, SSD storage |
